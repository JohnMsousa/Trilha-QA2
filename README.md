# Resumo da Aula

**Nesta aula, abordamos os seguintes tópicos:**

- **Refatoração de código para sorteio de números:**  
  Aprimoramos a função de sorteio do número secreto, tornando-a mais robusta e garantindo que o número sorteado não seja repetido.

- **Criação e manipulação de listas em JavaScript:**  
  Realizamos diversas atividades práticas que incluíram:  
  - Criação de uma lista vazia chamada `listaGenerica`.  
  - Criação de uma lista chamada `linguagensDeProgramacao` com os elementos: `'JavaScript'`, `'C'`, `'C++'`, `'Kotlin'` e `'Python'`.  
  - Adição dos elementos `'Java'`, `'Ruby'` e `'GoLang'` à lista `linguagensDeProgramacao` utilizando o método `.push()`.  

- **Acesso a elementos específicos em listas:**  
  Desenvolvemos exemplos que:  
  - Exibem o primeiro elemento de uma lista chamada `usuarios`.  
  - Exibem o segundo elemento de uma lista chamada `objetos`.  
  - Exibem o último elemento de uma lista chamada `times` utilizando o comprimento da lista (`.length - 1`).  

Esses tópicos reforçaram os conceitos fundamentais de manipulação de arrays e lógica de programação em JavaScript.
